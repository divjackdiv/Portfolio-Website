About this game

    This game was created as part of the GLASS in the summer 2016, the team was comprised of five game designer, and one developer (me). The game is a 2d Tower Defense about bacteria and antibiotics, you can get more info (and the instructions on how to play) about the game on the itch.io.You can also check out all my code for this game on my github repo.
