Shape Shifters

About this game

    This game is my current, and most ambitious project. It is a 2d strategic rpg set an world inspired by asian fables. The gameplay allows the player to create their own spells in a very flexible manner so that each player may battle in the way best suited to them. There is a quite a lot of code for this project which can be seen on the github repo, but the game is still in it's early stages.
